# FTZ Training Level1
=====================
  
  
## 프롬프트
* [trainer1@ftz trainer1]$
  - trainer1(앞쪽) -> 접속 ID
  - ftz -> 서버
  - trainer1(뒷쪽) -> 현재 위치
  
## ls
* 디렉토리와 파일 리스트를 출력
  * 옵션
    - l : 권한 Permission 용량 날짜 파일/디렉토리 명
    - a : 숨겨진 파일 표시
    
  * 파일은 -으로 시작, 디렉토리는 d로 시작
  * 숨겨진 파일은 이름이 .으로 시작
